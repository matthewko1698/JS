Inner Join

The inner join keyword takes two tables and joins them with a given condition. The given condition provides the selection when there is a commonality between both of the tables.

  1. Provide selection
  2. Provide FROM "first table"
  3. JOIN (inner) to "second table"
  4. ON clause giving two columns in each table and a statement
  5. Returns selection when the clause is true

A join is dependant on the similarities between two tables and more specifically columns in each row. If two column elements from two separate rows are equal from two different tables are equal, then the selection can be returned. The join can be a self join as well so the two tables are just the same table by different names. The records need to be part of both tables for any information to be returned unlike the left, right, or full joins. There is not a need to display what needs to be equal for an inner join. The equality statement gives a more precise selection of information.
